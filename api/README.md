## 📖 About

MyContacts back-end.

API to store contacts, using Node.js with Express and Postgres with Docker.

Contacts and categories CRUD.

## 🔥 Run

```bash

# install deps
$ cd api
$ yarn

# run and create containers
$ docker-compose up

# run the container with database
$ docker start pg

# run API
$ yarn dev
```
